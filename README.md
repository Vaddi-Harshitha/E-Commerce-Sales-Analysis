# E-Commerce Sales Analysis Project Overview

# This project analyzes e-commerce sales data using Python, Pandas, Matplotlib, and Machine Learning.

# Features

- Revenue Analysis
- Profit Analysis
- Category Analysis
- City Analysis
- Product Analysis
- Data Visualization
- Revenue Prediction using Linear Regression

# Technologies Used

- Python
- Pandas
- Matplotlib
- Scikit-learn

# Machine Learning

Linear Regression is used to predict revenue based on quantity sold.
